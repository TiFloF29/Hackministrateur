## Le dojo attaqué

Le OSINT Dojo a été attaqué, et a reçu une image du cybercriminel :

![Le dojo attaqué](https://raw.githubusercontent.com/OsintDojo/public/3f178408909bc1aae7ea2f51126984a8813b0901/sakurapwnedletter.svg)

L'analyse des données [EXIF](https://fr.wikipedia.org/wiki/Exchangeable_image_file_format) de l'image permet de trouver un nom d'utilisateur.

```bash
exiftool sakurapwnedletter.svg
```

## Reconnaissance

Une recherche du pseudonyme permet de trouver un compte [Github](https://github.com/sakurasnowangelaiko) et un compte [X/Twitter](x.com/sakuraloveraiko)

{% include elements/figure_spoil.html image="images/THM/20251115/Capture_ecran_2025-11-15_Recherche1.png" caption="Recherche du pseudonyme dans Duckduckgo" %}

Une clé publique {% include dictionary.html word="GPG" %} a été déposée sur Github dans le dossier PGP de l'utilisateur.

{% include elements/figure_spoil.html image="images/THM/20251115/Capture_ecran_2025-11-15_GPG.png" caption="Une clé GPG déposée sur Github" %}

Après avoir récupérer la clé, il est possible de trouver l'adresse mail de la personne.

```bash
gpg --show-keys cle.asc
```

## Lever le voile

L'analyse des dépôts de l'utilisateur, et particulièrement celui intitulé "ETH" dévoile qu'il possède un portefeuille de cryptomonnaie pour l'[Ethereum](https://fr.wikipedia.org/wiki/Ethereum).

Bien que le portefeuille n’apparaisse pas à première vue, une analyse de l'historique du fichier révèle des informations cruciales.

{% include elements/figure_spoil.html image="images/THM/20251115/Capture_ecran_2025-11-15_ETH.png" caption="Informations sur le portefeuille de cryptomonnaie" %}

En utilisant le site [coinstats.app](https://coinstats.app) avec l'identifiant du wallet trouvé, il est possible d'analyser l'historique des transactions du compte

{% include elements/figure_spoil.html image="images/THM/20251115/Capture_ecran_2025-11-15_ETH-Wallet.png" caption="Investigations sur les transactions" %}

Le lien disponible au niveau du hash permet d'obtenir plus d'informations sur le site [etherscan.io] concernant l'origine des transactions.

{% include elements/figure_spoil.html image="images/THM/20251115/Capture_ecran_2025-11-15_ETH-Transaction.png" caption="Source des versements" %}

Le site révèle également des transactions dans une autre devise UTSD qui correspond au **Tether**.

## Moquerie

Alors que les investigations suivent leur cours, le cybercriminel envoie un message se moquant des efforts déployés pour le retrouver.

![Message de moquerie](https://raw.githubusercontent.com/OsintDojo/public/main/taunt.png)

En enquêtant sur le premier compte Twitter trouvé au début des recherches, l'utilisateur indique qu'il ne se soucie plus d'oublier ses mots de passes, et semble avoir laisser un indice sur l'outil utiliser, DeepPaste, un site accessible depuis le Dark Web permettant de stocker toute sorte d'informations.

{% include elements/figure_spoil.html image="images/THM/20251115/Capture_ecran_2025-11-15_indicetwitter.png" caption="Le cybercriminel n'a peur de rien" %}

>Le site ne semble plus accessible depuis un moment, et le système d'indice de TryHackMe ayant changé, il n'est plus possible de répondre à cette question de façon légitime, pas même avec un coup de main de l'assistant. Assistant qui devrait nous donner le lien : https://raw.githubusercontent.com/OsintDojo/public/main/deeppaste.png

En recherchant l'identifiant md5 présent dans la seconde capture partagée par l'utilisateur dans le site DeepPaste, l'identifiant et le mot de passe de son réseau WiFi apparaissent en clair.

{% include elements/figure_spoil.html image="https://raw.githubusercontent.com/OsintDojo/public/main/deeppaste.png" caption="Informations sur les réseaux WiFi utilisés par le cybercriminel" %}

Une rapide recherche de type "find BSSID" permet de trouver le site [wigle.net](https://wigle.net/) capable de retrouver un BSSID à partir d'un SSID (DK1F-G ici)

{% include elements/figure_spoil.html image="images/THM/20251115/Capture_ecran_2025-11-15_Wigle.png" caption="Informations sur le réseau WiFi" %}
